## Why this exists

Open-source code licenses (MIT in our case) cover **copyright** — the right to copy and modify the source. They do **not** cover **trademarks** — the right to use the name and brand. Without a separate trademark notice, a fork could legally call itself "elicify-vertex Enterprise" and confuse users about who's behind it. This page makes the boundary explicit.

The pattern is standard across the open-source ecosystem: Linux, Kubernetes, Mozilla Firefox, MariaDB, and many others use the same dual structure (open-source code license + reserved trademark).
