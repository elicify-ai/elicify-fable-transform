# Getting help with elicify-vertex

A short routing guide. Pick the channel that matches what you're trying to do.

## I found a bug

→ Open a [GitHub issue](https://github.com/elicify-ai/elicify-vertex/issues/new/choose). Include:

- Package version (`npm ls @elicify-ai/elicify-vertex` or the version in `package.json`)
- OpenCode version
- OS
- How the plugin is loaded (`opencode.json` plugin entry — package name or `file://` path)
- Reproduction steps
- Relevant lines from `~/.config/opencode/.vertex-debug.log` if `VERTEX_DEBUG=1` was set
- Relevant events from `$VERTEX_DATA/.vertex-events.jsonl` (default under `~/.config/opencode/`)

## I have a feature idea

→ Open a [GitHub issue](https://github.com/elicify-ai/elicify-vertex/issues/new). For substantial features, discuss the design in an issue **before** writing code — saves wasted effort.

## I have a question — how do I use X?

→ [GitHub Discussions](https://github.com/elicify-ai/elicify-vertex/discussions) if enabled, otherwise open an issue with the `question` label. Start with the [README](README.md) and [docs index](docs/README.md).

## I found a security vulnerability

→ **Do not** open a public issue. See [`SECURITY.md`](SECURITY.md) for the private disclosure process. Either open a [private security advisory](https://github.com/elicify-ai/elicify-vertex/security/advisories/new) or email **connect@elicify.ai** with subject line `[security]`.

## I want to use the elicify-vertex name / brand

→ See [`TRADEMARKS.md`](TRADEMARKS.md). Most use cases (articles, integrations, unmodified redistribution) don't need permission. Commercial use of the brand does — email **connect@elicify.ai**.

## I want to contribute code

→ [`CONTRIBUTING.md`](CONTRIBUTING.md) for development setup and workflow. External contributors need a one-time [CLA](CLA.md) before their first PR can merge.

## I want to sponsor / fund the project

→ See [`.github/FUNDING.yml`](.github/FUNDING.yml) for current sponsorship channels (when enabled).

## Anything else (partnerships, press, commercial inquiries)

→ **connect@elicify.ai**
